### Функционал

- [ ] **Время работы кафе** — кнопка заказа неактивна, если кафе закрыто
- [ ] **Минимальная сумма заказа** с предупреждением в корзине

- [ ] **Стоимость и время доставки** по адресу (хотя бы фиксированные)

- [ ] **Темы и быстрый брендинг** — чтобы под нового клиента менять цвета/шрифты/название в одном файле

## Следующий проект: Барбершоп

**Концепция:** Telegram Mini App для записи в барбершоп.

### Чем отличается от кофейни (важно для портфолио — нужно показать разные UI-паттерны):

| Кофейня | Барбершоп |
|---|---|
| Корзина (можно купить N штук) | Запись на конкретное время (одна услуга = один слот) |
| Доставка по адресу | Гость приезжает сам |
| Чек оформляется один раз | Возможны напоминания и повторные визиты |
| Меню = каталог товаров | Меню = каталог услуг + мастера |
| Оплата вперёд (опционально) | Оплата на месте, бронь бесплатна |

### Экраны Барбершопа:

1. **Главная** — название барбершопа, премиум-эстетика (тёмные тона, золото/медь, грубая типографика)
2. **Услуги** — список с длительностью и ценой (Стрижка 1500₽/40мин, Борода 800₽/20мин, Комплекс 2000₽/60мин)
3. **Мастера** — карточки с фото, специализацией, рейтингом
4. **Календарь** — выбор даты (горизонтальная лента дней)
5. **Слоты времени** — выбор времени, занятые слоты неактивны
6. **Подтверждение** — имя, телефон, итоговая запись
7. **Успех** — «Записан на 5 июня 14:00 к Артёму»

### Технически новое в проекте (отличия от кофейни):

- Календарь дат
- Сетка временных слотов с занятостью
- Карточки мастеров с фото
- Логика «занято / свободно» (хотя бы захардкодить пока)
- Тёмная мужская тема (вместо тёплой коричневой)

### Эстетика:

- Тёмный фон (антрацит/уголь)
- Акцент — медь/золото/охра
- Шрифты — что-то «брутальное» (Bebas Neue, Anton, Oswald) для заголовков + чистый sans для тела
- Фото с фактурой (бритвы, ножницы, кожа, тёплый свет)

## Как начать следующую сессию (важно!)

Когда откроешь новый чат с ИИ в Cursor, начни так:

> «Привет. Открой файл `IDEAS.md` в корне проекта — там весь контекст. Сегодня хочу заняться [что именно из списка].»

ИИ прочитает этот файл и моментально войдёт в контекст. Не придётся пересказывать всю историю проекта.
